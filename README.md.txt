El Survey Form es un formulario diseñado como parte de un proyecto de reclutamiento y de aprendizaje en desarrollo web. 
Este formulario tiene como objetivo recopilar información de los usuarios de manera estructurada y accesible. 
Demuestra habilidades en HTML, CSS y diseño web responsivo, y sigue las mejores prácticas para la creación de 
formularios interactivos y fáciles de usar. El formulario está diseñado para ser fácil de usar tanto en 
dispositivos de escritorio como móviles, asegurando una experiencia de usuario óptima.

Características
	-Diseño Responsivo: El formulario está diseñado para adaptarse a diferentes tamaños de pantalla, desde dispositivos móviles hasta monitores de escritorio.
	-Validación de Entradas: Los campos del formulario incluyen validaciones para asegurar que los datos ingresados sean correctos y completos antes de ser enviados.
	-Interfaz Amigable: Uso de un diseño limpio y moderno que facilita la navegación y el uso del formulario.
	-Accesibilidad: Se han seguido buenas prácticas para asegurar que el formulario sea accesible para todos los usuarios, incluyendo aquellos con discapacidades.
Estructura del Proyecto
	-index.html: El archivo principal que contiene la estructura del formulario.
	-styles.css: Archivo de estilos CSS que define la apariencia y diseño del formulario.
	-README.md: Documento que describe el propósito y las características del proyecto.
Tecnologías Utilizadas
	-HTML5: Para la estructura del formulario.
	-CSS3: Para el diseño y la disposición visual del formulario.
	
Como usar?
1. Descargar o Clonar el Repositorio:
	- Puede clonar el repositorio usando el siguiente comando:
		"git clone https://github.com/tu-usuario/survey-form.git"

2. Abrir el Navegador:
	- Simplemente abre el archivo 'index.html' en tu navegador web preferido.
